Searching

The reddit code on which Sciteit is based uses indextank to power the majority of their searches. Unfortunately, indextank no longer takes new registrations and as such a new search provider has had to be implemented (solr, which reddit used before it switched to indextank). This is working but there are a few limitations currently. The "Advanced search" parameters shouldn't do anything. Essentially you're limited to two types of search

1. Search for links. This will search for links (posted content) that has a title or author matching your query. The search can be restricted to a given subsciteit by clicking the appropriate box. This is the function of the main search box.

2. Search for a subsciteit. As the name suggests, this can be used to find a subsciteit you are looking for. This search can be accessed by clicking on the subsciteit button in help.

If something you just added isn't showing up on search, that's because it hasn't been indexed yet. Please be patient, the indexer will get to it in a few minutes.
