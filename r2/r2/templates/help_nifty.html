Nifty features you may not have known about

This is a list of cool things that not all users seem to know about. Please send us feedback if you can think of ways to make these features more discoverable.

Misc

  • When a submission's number is colored, it means it's reached the #1 spot on sciteit. Clicking on this number will take you to a running list of all the #1 sciteit links.
  • Both the sciteit toolbar and bookmarklets have a "serendipity!" button, which takes you to a random link in the sciteit queue. You can also go to http://sciteit.com/r/[category]/random to get a random link from a particular category.
  • You can see the most recent comments across all of sciteit by visiting sciteit.com/comments. This works for subsciteits too, by going to /r/(subsciteit)/comments .
  • A dotted outline around a headline indicates a link you just visited. This makes it easier to up-vote, down-vote or comment on the submission once you return to Sciteit.

Pseudosciteits and other URL tricks

  • You can view posts from multiple sciteits by separating them with '+' characters in the URL, such as sciteit.com/r/bugs+ideasfortheadmins
  • To view posts from all sciteits, use /r/all
  • To see all posts within a given domain, go to http://sciteit.com/domain/[domain] -- like sciteit.com/domain/nature.com.
  • To change the number of posts per page without logging in, for instance when you're using an RSS reader, add a "limit" parameter to the query string. For example sciteit.com/.rss?limit=42.

The toolbar

  • Toolbar links can be autogenerated by prepending http://sciteit.com/ to any URL. For example, if you're visiting the Wikipedia page on the Konami code (http://en.wikipedia.org/wiki/Konami_Code) and you're wondering if it was ever submitted to sciteit, just go to http://sciteit.com/http://en.wikipedia.org/wiki/Konami_Code (you can leave off either or both "http://"s). ##
  • You can also get a bookmarklet to do the above for you.
  • If you do the above, and the link hasn't been submitted, you'll be taken to the submit page. So this is a great shortcut for submitting links to sciteit.
  • If you visit http://sciteit.com/[some submission id], you'll get taken to the submission.

Auto population from RSS feeds

If you are the moderator of a category, you will be given the option to auto-populate your category with links from an RSS feed. This will work fine as long as the RSS feed you use has entries that have both a 'link' and 'title' tag that are set to something sensible. This should be the case from pretty much any RSS feed.

Most major journals and pre-print servers have a suitable RSS feed. For example ArXiv provides an RSS feed for every category at http://arxiv.org/rss/CATEGORY-NAME.

The RSS feeds are updated every morning at 3am GMT. New links will not be added if any of the following are true.

  • The link has already been submitted somewhere else.
  • Another category is using the same RSS feed. First in best dressed I'm afraid.
  • The title or link fields are missing or have invalid data. Note titles are truncated to 300 characters.
