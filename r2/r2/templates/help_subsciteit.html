Categories, "sciteits" or "subsciteits"

Every post on Sciteit belongs to a category. These help to organize content by topic area so that it is easier for users to find exactly what they're looking for. These categories are also called "subsciteits" or sometimes just "sciteits". Don't be confused, they are all just different names for the same thing. Just think of them as categories and you'll be fine.

The categories are organized in a tree structure with the sciteit.com category sitting at the top. Each category will have a parent category. It may also have children that it inherets from. Anything posted to a child category will show up in the parent category. Additionally, each category might also have siblings, which are just categories that share a parent. This category tree can be easy navigated using the topbar, which will list the parent category in bold, the current category in purple, siblings as normal and children in gray.

Any registered user can create a new category, by clicking here. A link to this page will appear in the sidebar for any users that have been registered for more than 30 days. On the category creation page, you will be asked to enter the following information:

  • A name for your category. Spaces are not allowed. Must be unique.

  • A title for your category. This is a short description of your category, that will appear in the titlebar of the browser (amongst other places).

  • A parent for your category. Each new category must have a parent category. For example if you are making a category for astrophysics, you might select physics as the parent category. Note that only administrators can create categories that have the root category as a parent.

  • A more detailed description of your category (500 characters). This will appear in the sidebar when users visit your category.

  • Select a language for your category. I can't guarantee anything other than english will work at the moment.

  • The type of category. This determines who will be able to view and submit to your category. It is possible to have completely private categories that are only visible to a set of approved members, for the category to be visible to anyone but only certain people able to submit, or completely open.

  • Should links, text posts or both be allowed?

  • Can this category be one of the default options for users who haven't logged in?

  • Should a thumbnail of content be displayed?

  • You will be given the option of entering a domain. If you specify one here, websites hosted at this domain will be able to display this category on their website.

  • You will be given the option of entering a RSS feed. If you choose to enter one, this feed will be used to automatically add new links each day.

The user who creates a category or sciteit will automatically be made a moderator of that category. More information about being a moderator can be found here.
